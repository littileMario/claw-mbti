# 🦞 claw-mbti — 龙虾 MBTI 性格诊断

基于你和龙虾的日常互动，你的龙虾会告诉你它是什么性格。

不需要做题，不需要问卷——龙虾默默观察着你，然后告诉你它是谁。

## 一键安装（也可用于更新）
步骤1，把以下这行代码复制下来，贴到龙虾的对话框里：
```bash
[ -d ~/.openclaw/workspace/skills/claw-mbti ] && cd ~/.openclaw/workspace/skills/claw-mbti && git pull || git clone https://github.com/xiaoyang0807/claw-mbti.git ~/.openclaw/workspace/skills/claw-mbti
```

步骤2，安装完成后，对龙虾说：

> 测测你的 MBTI

## 诊断原理

传统 MBTI 测试靠你回答问题来判断——但你对自己的认知可能有偏差。

龙虾诊断不一样。它分析的是你**真实的行为模式**：

| 维度 | 分析依据 | 示例 |
|------|----------|------|
| E/I | 互动密度与话题来源 | 密集深聊内心 → I 倾向 |
| S/N | 你问龙虾的问题类型 | "如果龙虾能飞会怎样" → N 倾向 |
| T/F | 你追问龙虾时的风格 | "有什么数据支撑吗" → T 倾向 |
| J/P | 你的互动是有计划还是随性的 | 话题频繁跳转 → P 倾向 |

追问时充满好奇心和发散性思考，是典型的 ENFP 特征。
