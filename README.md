# 🦞 claw-mbti — 龙虾 MBTI 性格诊断

基于你和龙虾的日常互动，诊断你是哪种 MBTI 型的龙虾。

不需要做题，不需要问卷——龙虾默默观察着你。

## 一键安装

```bash
git clone https://github.com/littileMario/claw-mbti.git ~/.openclaw/workspace/skills/claw-mbti
```

安装完成后，对龙虾说：

> 测测我的 MBTI

## 诊断原理

传统 MBTI 测试靠你回答问题来判断——但你对自己的认知可能有偏差。

龙虾诊断不一样。它分析的是你**真实的行为模式**：

| 维度 | 分析依据 | 示例 |
|------|----------|------|
| E/I | 你和龙虾聊天的频率和热情度 | 日均 5 次对话 → E 倾向 |
| S/N | 你问龙虾的问题类型 | "如果龙虾能飞会怎样" → N 倾向 |
| T/F | 你追问龙虾时的风格 | "有什么数据支撑吗" → T 倾向 |
| J/P | 你的互动是有计划还是随性的 | 话题频繁跳转 → P 倾向 |

## 输出示例

```
我是 ENFP型 的龙虾

```
